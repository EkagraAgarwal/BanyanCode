Search code graph nodes using semantic embedding similarity.

When an embedding model is configured (BANYANCODE_EMBEDDING_MODEL env var), uses cosine similarity to find semantically similar code. Falls back to keyword matching when no embedding model is available.

Arguments:
- query: Search query string
- limit (optional): Maximum number of results (default: 10)
- fileGlob (optional): Filter to files matching glob pattern

Returns:
- hits: Array of { node, score } sorted by relevance
- degraded: true when embeddings unavailable, false otherwise
